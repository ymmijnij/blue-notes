# Challenge da Casa - Medio - Bonobo Studio
Sfida tecnica React. Creare una Notes App.
## Installazione
```
npm run build
npm install -g serve
serve -s build
```
Navigare a `http://localhost:3000/` su sulla macchina locale per utilizzare l'app.
